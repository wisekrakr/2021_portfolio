wise_filetransfer:$_

This java app works with certificates. The server (Certificate Authority) will ask for the client's private key (within a certificate) as authentication. This app will encrypt and decrypt the files based of of these keys. Start sending!

securecomm:$_

Working with certificates has many avenues, so I tried again, this time creating SSL connections, between multiple clients. This project includes an authentication protocol, encrypted messages (including audio messages), protocol buffers, multithreading and multiplexing, and much more. Get secure and come together!
